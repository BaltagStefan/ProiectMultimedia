# Model 3D

Frontend-ul include un model procedural performant, folosit automat ca fallback.

Pentru un model auto real, adăugați în acest director fișierul:

`car.glb`

Nodurile recomandate pentru selecția viitoare prin raycasting sunt:

- `Hood`
- `Engine`
- `Wheels`
- `Brakes`
- `Headlights`
- `Trunk`
- `Battery`
- `Interior`
